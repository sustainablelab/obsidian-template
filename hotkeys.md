# Open another vault

`Ctrl`+`Shift`+`O`

# Select active editor pane

- focus-left
    - `Alt`+`ArrowLeft`
- focus-right
    - `Alt`+`ArrowRight`
- focus-top
    - `Alt`+`ArrowUp`
- focus-bottom
    - `Alt`+`ArrowDown`

# Navigate backward and forward

- go backward
    - `Ctrl`+`Alt`+`H`
- go forward
    - `Ctrl`+`Alt`+`L`

# Toggle sidebars

- toggle *backlinks* and *tags* sidebar
    - `Ctrl`+`Alt`+`ArrowRight`
    - open *tags* pane
        - `Ctrl`+`Shift`+`T`
    - open *backlinks* pane
        - `Ctrl`+`Shift`+`B`
- toggle *file-explorer* and *search*
    - `Ctrl`+`Alt`+`ArrowLeft`

# Folds

- unfold all
    - `Ctrl`+`R`
- fold all
    - `Ctrl`+`M`
